## 核心要点

### 8 周课程精华

1. **第一周**：AI 编码工具全貌 - Cursor、Claude Code、NotebookLM 入门，掌握 API 接入基础
2. **第二周**：深入 Cursor - 人机交互式智能编程，Rules 设置与多模态交互
3. **第三周**：Claude Code Agent 驱动开发 - 任务分解、协作与代码审查
4. **第四周**：Spec Driven 开发流程 - 从想法到产品的完整工作流
5. **第五周**：MCP 协议深入 - 构建自定义 MCP Server
6. **第六周**：Agent 核心原则 - 逆向工程与 Multi-Agent 协作
7. **第七周**：多智能体系统 - 架构设计与团队编排
8. **第八周**：未来展望 - Vibe Coding、人机协作与职业发展

### AI 时代开发者的关键洞察

- **Vibe Coding** - 以自然语言描述意图，英语成为新的编程语言
- **人机协作** - 从「驾驶员」转变为「调度员」，掌握指挥棒而非方向盘
- **经济学驱动** - 沉没成本消解，快速失败快速构建成为新常态
- **职业转型** - 资深开发者转向架构师，初级开发者以 AI 为导师
